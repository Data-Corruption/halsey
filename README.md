# Halsey [![Build / Release](https://github.com/Data-Corruption/halsey/actions/workflows/build.yml/badge.svg)](https://github.com/Data-Corruption/halsey/actions/workflows/build.yml) ![License](https://img.shields.io/github/license/Data-Corruption/halsey)

Go-based Discord bot for link expansion and lightweight archiving. Minimal footprint, careful with rate limits, and designed for small private servers. Setup assumes developer experience; this isn’t intended for broad distribution. 

```sh
curl -fsSL https://raw.githubusercontent.com/Data-Corruption/halsey/main/scripts/install.sh | sh
```
